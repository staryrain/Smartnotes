# Smartnotes v1.0.0 Release Notes

## 版本信息
- **版本**: 1.0.0
- **发布日期**: 2026-02-27
- **构建平台**: Windows (x64)

## 主要功能
- **每日任务**: 快速记录和管理每日待办事项。
- **长期目标**: 设定并追踪长期目标，支持分解为子任务。
- **成就系统**: 记录已完成的重要里程碑。
- **计划管理**: 提前规划未来任务。
- **桌面便签**: 支持穿透模式和透明度调节，完美融入桌面环境。

## 修复与改进
- **安全性**: 增强了 IPC 通信的安全性，防止潜在的恶意调用。
- **稳定性**: 修复了数据库操作中的潜在崩溃问题，增加了全局错误捕获。
- **性能**: 优化了重复任务生成的查询逻辑，减少了数据库负载。
- **数据一致性**: 为关键数据操作添加了事务支持。

## 安装与运行
### 绿色版 (Portable)
由于构建环境限制，本次发布提供解压即用的绿色版：
1. 进入 `dist_build/win-unpacked` 目录。
2. 双击 `Smartnotes.exe` 即可启动。
3. 您可以创建该文件的快捷方式到桌面。

### 注意事项
- 首次运行时，Windows Defender 可能会提示“未知发布者”，这是因为应用程序未购买数字证书签名。请点击“更多信息” -> “仍要运行”。
- 数据文件默认存储在用户目录下，升级版本时请注意备份。
